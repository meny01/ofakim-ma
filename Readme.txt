לבניית התוכנית השתמשתי ב 
asp.net core 3.1
מבחינת הארכיטקטורה השתמשתי ב 
MVC 
עם שכבה נוספת לפישוט והסתרת המידע מהקונטרולר, 
עבור כל השירותים השתמשי ב 
dependency injection
שעובד מאוד טוב ויעיל עם דוט נט קור

עבור קבלת המידע של ההמרה השתמשתי באיפיאיי חינמי,  את השירות עבור ה איפיאיי רשמתי ב
CurrencyConverterApi.cs

עבור אחסון ומשיכת המידע השתמשתי ב
SQL server
.מבחינת היעילות, פה ספציפית היה עדיף להשתמש בקובץ כי אין הרבה מידע ולא צריך לעשות  מניפולציות על המידע
אבל אם מסתכלים לטווח הארוך  אם נרצה לבצע שינויים ,לשנות או להוסיף המון מידע כדאי להשתמש ב 
SQL server

עבור הגדרת הדתה בייס השתמשתי בשלוש הרחבות מנוגט
Microsoft.EntityFrameworkCore - שימוש בלינק והגדרת כל השירותים הדרושים לשימוש בדתה בייס
Microsoft.EntityFrameworkCore.SqlServer - חיבור הסרבר לדתה בייס 
Microsoft.EntityFrameworkCore.Tools - הוספת שירותים כגון add-migration,update-database....

code first - add-migration,update database
